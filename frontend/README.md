## Getting Started

Add the `.env` file in root directory

```.env
PORT =
MONGODB_URI = 
NEXT_PUBLIC_SERVER_BASE_ADDR =
JWT_SECRET =
```


Then, run the development server:

```bash
pnpm run dev
```
